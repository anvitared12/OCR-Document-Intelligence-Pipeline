To solve this run the command - "python -m uvicorn app:app --reload" instead of uvicorn app:app --reload , this This forces it to run inside whichever Python the python command resolves to (3.11, where your packages actually are) instead of whatever separate uvicorn.exe happens to be first on your PATH.
